If enabled, Jenkins will allow showing dangerous permissions in the Web UI. The behavior depends on the value of the org.jenkinsci.plugins.rolestrategy.permissions.DangerousPermissionHandlingMode.enableDangerousPermissions system property.
  • If the property is true, the permissions will be always shown
  • If the property is false or undefined, the permissions will be shown depending on the setting. But these permissions will take no effect.
